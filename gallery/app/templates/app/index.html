Welcome to Text-Based Image Manipulator

Text-Based Image Manipulator is a novel image editing tool. Using this tool, users can alter the contents of an image without changing the text-irrelevant content such as the background. Built using Generative Adversarial Networks with Text-Adaptive Discriminators, Bilinear Residual Layers and several other state-of-the-art Deep Learning techniques, this tool produces remarkable results.
